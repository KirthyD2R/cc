  input_pdfs/invoices/          → 01, 02
  output/extracted_invoices.json → 02, 03
  output/bill_results.json       → 03, 05
  output/cc_transactions.json    → 04, 05

  None use environment variables or a central path config.

  ---
  MEDIUM: Deprecated Dependencies

  requirements.txt uses fuzzywuzzy (deprecated) instead of thefuzz. There's even a comment about migrating but it hasn't been done.

  OCR dependencies (pytesseract, pdf2image) are commented out — scanned PDFs will crash.

  ---
  MEDIUM: Undocumented Script Interfaces

  run_loop.py dynamically imports and calls run() from each script with specific parameter signatures (e.g., run(since_timestamp=..., known_email_ids=...,
  headless=True)). These signatures are not validated or documented — if any script changes its signature, the loop breaks silently.
